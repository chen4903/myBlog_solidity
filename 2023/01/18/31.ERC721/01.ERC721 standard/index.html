01.ERC721 standard

发表于2023-01-18|更新于2023-01-18|31.ERC721

ERC721 standard

digest

  • NFT,Non-Fungible Token
  • 通常使用在艺术品、房子、虚拟收藏品等不可替代的物品中

the smallest uint

9 functions

  • 获取分配给owner的所有NFTs数量
  • 查询拥有token ID为_tokenId的NFT的所属者的owner地址
  • 将token ID号为_tokenId的NFT从_from地址的用户转移到_to地址的用户(需要加上条件判断)
  • 将token ID号为_tokenId的NFT从_from地址的用户转移到_to地址的用户，并携带数据data(_from和_to不可为合约地址)
  • 将token ID号为_tokenId的NFT从_from地址的用户转移到_to地址的用户(_to不可为合约地址)
  • owner授权
  • 获取_tokenId代币对应的拥有者的地址address
  • 授权给第三方_operator进行管理个人的ERC721代币token资产
  • 查看_owner是否授权_operator管理所有的ERC721代币的token资产

3 events

  • NFTs资产转移事件
  • NFTs资产授权地址发生变更事件
  • 所有NFTs资产被授权事件

optional

ERC721Metadata

This allows your smart contract to be interrogated for its name and for details about the assets which your NFTs represent.

1.描述NFTs收藏品的名称

2.合约中NFTs称

3.给定资产唯一资源标识符(URL)对应的详细信息

ERC721Enumerable

This allows your contract to publish its full list of NFTs and make them discoverable.

1.追踪计数智能合约中的NFTs数量

2.枚举有效的NFT

3.枚举分配给拥有者owner懂得NFT

interface ERC165

  • Query if a contract implements an interface

interface ERC721TokenReceiver

  • A wallet/broker/auction application MUST implement the wallet interface if it will accept safe transfers.
