02.hardhat查询链上数据

我们知道，区块链是公开透明的，那么合约中的任何数据都是可以查询的，即使他是private。

查询非引用类型

这里用一个例子说明如何查询一个合约中的private属性。我们的目标：查询private属性。假如我们需要查询的数据是uint，string等非引用类型：

  • 使用hardhat进行查询

  • 在hardhat.config.js输入配置信息：goerli测试网络

    • ```js
      const GOERLI_API_URL = “https://eth-goerli.g.alchemy.com/v2/AaEomr1g_2UC3GX_7J4XbxvvjKtn6UAi";
      const PRIVATE_KEY = “??????????????????????????????????????????????”;

      module.exports = {
      solidity: “0.8.9”,
      networks: {
      goerli: {
      url: ${GOERLI_API_URL},
      accounts: [${PRIVATE_KEY}]
      }
      },
      };

工具：将合约地址作为参数输入，得到结果：0xba9c4669505d41dc0aadb354d6e0539f711569beeb21c63d30f31e5073ed2bc1

查询得到结果（网络不好，需要试过很多次才成功）。得到数据，a转化为十进制就是10。
