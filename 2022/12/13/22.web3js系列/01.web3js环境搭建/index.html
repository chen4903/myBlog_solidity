web3js环境搭建

web3js概述

  • 开发以太坊区块链应用程序，涉及到以下部分

    • 智能合约开发–使用solidity语言编写代码，部署到区块链

    • 网站或客户端开发–与区块链中的智能合约进行交互，读写数据

    • 与Ajax与web服务器进行通信的机制类似，可以使用Web3.js读写以太坊区块链。

  • web3.js库是一个javascript库，是以太坊官方提供的和节点交互的JavaScript SDK (js-sdk ) ，可以帮助智能合约开发者使用HTTP或者IPC与本地的或者远程的以太坊节点进行交互（Web3是对JSON RPC请求的封装）。web3.js主要包含下面几个库

    • web3-eth用来与以太坊区块链和智能合约交互。【主要】
    • web3-shh 用来控制whisper 协议与p2p通信，以及广播。【基本用不到】
    • web3-bzz用来与swarm协议交互。
    • .web3-utils 包含了一些DApp开发应用的功能。【主要】

  • web3js中文文档

  • 通过web3.js库，可以连接任何暴露了RPC【RPC即网络通信】层的以太坊节点，可以让你通过节点执行很多与区块链进行交互的任务，例如︰

    • 以太币转账
    • 读写智能合约中的数据·创建智能合约
    • 发送交易

  • 区块链网络是一个点对点的对等网络，网络由各个节点相互连接构成，智能合约的代码与数据存储在区块链中。使用web3.js可以向区块链网络中某个以太坊节点发出请求，与区块链网络进行交互。此过程如下图所示:

web3js环境安装

  • 在Windows系统下安装nodejs及npm，下载安装包安装即可https: / / w w w.cnblogs.com/ hackyo/p/8110951.html
  • 安装“Code Runner”插件，就可以运行node.js的文件了
  • 在vScode的“终端”中执行下面代码，安装Web3.js: npm install web3。安装完成即可在vscode中开发web3js。

连接以太坊网络

  • 要访问以太坊网络，需要连接到Ethereum节点
  • 有几种方法可以做到这一点，可以使用Geth或Parity运行自己的Ethereum节点。但这需要从区块链下载大量数据并保持同步，很麻烦。比较方便的方法是使用Ganache太坊节点仿真器软件管理节点

使用Ganache

  • ganache -cli是以太坊节点仿真器软件ganache的命令行版本，可以方便开发者快速进行以太坊DApp的开发与测试。
  • ganache两个版本
    • 带有用户界面的图形应用程序
    • 命令行版本
