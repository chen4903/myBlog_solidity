04.returnе’Ңreturns

еҸ‘иЎЁдәҺ2022-11-27|жӣҙж–°дәҺ2022-11-27|02.solidityиҜӯжі•

returnе’Ңreturns

SolidityжңүдёӨдёӘе…ій”®еӯ—дёҺеҮҪж•°иҫ“еҮәзӣёе…іпјҡreturnе’ҢreturnsпјҢд»–д»¬зҡ„еҢәеҲ«еңЁдәҺпјҡ

  • returnsеҠ еңЁеҮҪж•°еҗҚеҗҺйқўпјҢз”ЁдәҺеЈ°жҳҺиҝ”еӣһзҡ„еҸҳйҮҸзұ»еһӢеҸҠеҸҳйҮҸеҗҚпјӣ
  • returnз”ЁдәҺеҮҪж•°дё»дҪ“дёӯпјҢиҝ”еӣһжҢҮе®ҡзҡ„еҸҳйҮҸгҖӮ

дёҠйқўиҝҷж®өд»Јз ҒдёӯпјҢжҲ‘д»¬еЈ°жҳҺдәҶreturnMultiple()еҮҪж•°е°ҶжңүеӨҡдёӘиҫ“еҮәпјҡreturns(uint256, bool, uint256[3] memory)пјҢжҺҘзқҖжҲ‘д»¬еңЁеҮҪж•°дё»дҪ“дёӯз”Ёreturn(1, true, [uint256(1),2,5])зЎ®е®ҡдәҶиҝ”еӣһеҖјгҖӮ

е‘ҪеҗҚејҸиҝ”еӣһ

жҲ‘д»¬еҸҜд»ҘеңЁreturnsдёӯж ҮжҳҺиҝ”еӣһеҸҳйҮҸзҡ„еҗҚз§°пјҢиҝҷж ·solidityдјҡиҮӘеҠЁз»ҷиҝҷдәӣеҸҳйҮҸеҲқе§ӢеҢ–пјҢе№¶дё”иҮӘеҠЁиҝ”еӣһиҝҷдәӣеҮҪж•°зҡ„еҖјпјҢдёҚйңҖиҰҒеҠ returnгҖӮ

еңЁдёҠйқўзҡ„д»Јз ҒдёӯпјҢжҲ‘д»¬з”Ёreturns(uint256 _number, bool _bool, uint256[3] memory _array)еЈ°жҳҺдәҶиҝ”еӣһеҸҳйҮҸзұ»еһӢд»ҘеҸҠеҸҳйҮҸеҗҚгҖӮиҝҷж ·пјҢжҲ‘д»¬еңЁдё»дҪ“дёӯеҸӘйңҖиҰҒз»ҷеҸҳйҮҸ_numberпјҢ_boolе’Ң_arrayиөӢеҖје°ұеҸҜд»ҘиҮӘеҠЁиҝ”еӣһдәҶгҖӮ

еҪ“з„¶пјҢдҪ д№ҹеҸҜд»ҘеңЁе‘ҪеҗҚејҸиҝ”еӣһдёӯз”ЁreturnжқҘиҝ”еӣһеҸҳйҮҸпјҡ

и§Јжһ„ејҸиөӢеҖј

solidityдҪҝз”Ёи§Јжһ„ејҸиөӢеҖјзҡ„и§„еҲҷпјҢж”ҜжҢҒиҜ»еҸ–еҮҪж•°зҡ„е…ЁйғЁжҲ–йғЁеҲҶиҝ”еӣһеҖјгҖӮ

  • иҜ»еҸ–жүҖжңүиҝ”еӣһеҖјпјҡеЈ°жҳҺеҸҳйҮҸпјҢе№¶дё”е°ҶиҰҒиөӢеҖјзҡ„еҸҳйҮҸз”Ё,йҡ”ејҖпјҢжҢүйЎәеәҸжҺ’еҲ—гҖӮ
  • иҜ»еҸ–йғЁеҲҶиҝ”еӣһеҖјпјҡеЈ°жҳҺиҰҒиҜ»еҸ–зҡ„иҝ”еӣһеҖјеҜ№еә”зҡ„еҸҳйҮҸпјҢдёҚиҜ»еҸ–зҡ„з•ҷз©әгҖӮдёӢйқўиҝҷж®өд»Јз ҒдёӯпјҢжҲ‘д»¬еҸӘиҜ»еҸ–_boolпјҢиҖҢдёҚиҜ»еҸ–иҝ”еӣһзҡ„_numberе’Ң_arrayпјҡ
